# 2048 AI - CPU & GPU Version

Ce projet est une Intelligence Artificielle qui joue automatiquement au jeu 2048.

Deux versions sont disponibles :
- `code_cpu.py` : version classique sur processeur (CPU)
- `code_gpu.py` : version accélérée avec CUDA (GPU)

---

## Dépendances

- Python 3
- numpy
- numba (pour la version GPU)

Installation rapide :

```bash
pip install numpy numba
```

---

## Lancer le programme

### Version CPU :
```bash
python code_cpu.py
```

### Version GPU :
```bash
python code_gpu.py
```
